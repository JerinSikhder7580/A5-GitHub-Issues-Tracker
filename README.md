## Difference between var, let and const

### var
* We can change the value of var ,but  there are some problem using var.So we do not use var.

### let
*We can change the value of let .It is safer  than var.So we usually use it .

### const 
*The value of const cannot be reassigned.Once a value is assigned , it cannot be changed.



## What is the spread operator (...)?

* The spread operator (...) is used to expand or spread the elements of an array or object into individual elements.

##  What is the difference between map(), filter(), and forEach()?


* Map()-> creates a new array by changing every element of the original array.
* filter()-> creates a new array with only the elements that pass a condition.
* forEach()-> runs a function for each elements ,but it does not create a new elements.
